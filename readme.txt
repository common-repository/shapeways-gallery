Displays a dynamic gallery of your Shapeways designs either as a widget, or in a post. 

== Description ==

This widget connects to your Shapeways account and displays a dynamic list of designs. It works in two modes:

1. Widget: just add the 'Shapeways Gallery' widget to your theme and configure it.

2. In a post: enter the following shortcode in a post:

[shapeways option=value ...]

Available options are:

* shop - the name of your shop
* shopsection - the name of a section in your shop (ignored if 'shop' hasn't been set).
* fav - display the favorites of this username
* cols, rows - the number of columns and rows in your widget. 
* style - allowed valued are 'light' and 'dark'. The default value is 'light'

For in-depth information on this widget, see the [widget tutorial page on Shapeways.com](http://www.shapeways.com/tutorials/widget).
